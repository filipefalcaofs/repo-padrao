# Stack Django

**Linguagem:** Python

Stack **custom** para projetos [Django](https://www.djangoproject.com/) + [Django REST Framework](https://www.django-rest-framework.org/).

## Conteúdo

| Item | Descrição |
|---|---|
| `django-core.mdc` | Models, views, admin, migrations |
| `django-drf.mdc` | API REST (serializers, viewsets) |
| `django-development` | Skill backend |
| `django-drf` | Skill DRF |
| `pytest-django` | Skill testes |
