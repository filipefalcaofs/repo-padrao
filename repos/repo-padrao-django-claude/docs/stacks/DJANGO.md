# Projetos Django com repo-padrao

> **Linguagem:** Python  
> **Clone direto:** [`repos/repo-padrao-django`](../../repos/repo-padrao-django)

Stack secundária **custom** — Django + DRF + pytest.

## Bootstrap

```bash
git clone https://github.com/filipefalcaofs/repo-padrao-django.git meu-app
cd meu-app
python -m venv .venv && source .venv/bin/activate
pip install django djangorestframework pytest-django
django-admin startproject config .
```
