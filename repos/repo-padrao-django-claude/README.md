# repo-padrao-django-claude

**IDE:** Claude Code · **Linguagem:** Python

Template **Django + DRF** — arquivos só para Claude Code.

## Clone

```bash
git clone https://github.com/filipefalcaofs/repo-padrao-django-claude.git meu-app
cd meu-app
rm -rf .git && git init
python -m venv .venv && source .venv/bin/activate
pip install django djangorestframework pytest-django
django-admin startproject config .
```

Ver [`docs/stacks/DJANGO.md`](docs/stacks/DJANGO.md).
