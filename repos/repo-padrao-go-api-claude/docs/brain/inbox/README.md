# inbox/ — Captura rápida (Capture)

Caixa de entrada do segundo cérebro. Tudo que chega entra aqui **sem fricção**: ideias, links, dúvidas, achados de pesquisa, comentários a revisar, screenshots, transcrições.

## Regras

- **Zero organização na captura.** Salvar primeiro, organizar depois.
- Nome livre, em `kebab-case.md`. Data no topo do arquivo, não no nome.
- Pode ter qualquer formato: bullet list, parágrafo solto, link com uma frase.
- **Esvaziar semanalmente.** A cada sessão de organização, mova cada item para a pasta PARA correta:
  - Tem prazo e objetivo? → `1-projects/`
  - Responsabilidade contínua? → `2-areas/`
  - Referência reutilizável? → `3-resources/`
  - Já não é mais relevante? → deletar ou `4-archive/`

## Template mínimo

```markdown
---
data: YYYY-MM-DD
fonte: <link, conversa, ideia própria, etc.>
---

<conteúdo bruto>
```

Não pense, capture. A organização vem depois.
