# repo-padrao-jhipster-react-cursor

**IDE:** Cursor · **Linguagem:** Java · **Frontend:** react

Template **JHipster + react** — arquivos só para Cursor.

## Clone

```bash
git clone https://github.com/filipefalcaofs/repo-padrao-jhipster-react-cursor.git meu-app
cd meu-app
rm -rf .git && git init
npm install -g generator-jhipster
jhipster
```

Ver [`docs/stacks/JHIPSTER.md`](docs/stacks/JHIPSTER.md).
