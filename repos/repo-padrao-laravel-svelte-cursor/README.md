# repo-padrao-laravel-svelte-cursor

**IDE:** Cursor · **Linguagem:** PHP · **Frontend:** Inertia svelte

Template **Laravel + Inertia (svelte)** — arquivos só para Cursor.

## Clone

```bash
git clone https://github.com/filipefalcaofs/repo-padrao-laravel-svelte-cursor.git meu-app
cd meu-app
rm -rf .git && git init
composer create-project laravel/laravel .
composer require laravel/boost --dev
php artisan boost:install
```

Ver [`docs/stacks/LARAVEL.md`](docs/stacks/LARAVEL.md).

Habilitar `laravel-boost` no MCP Settings (Cursor).
