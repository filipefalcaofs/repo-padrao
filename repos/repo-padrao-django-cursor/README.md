# repo-padrao-django-cursor

**IDE:** Cursor · **Linguagem:** Python

Template **Django + DRF** — arquivos só para Cursor.

## Clone

```bash
git clone https://github.com/filipefalcaofs/repo-padrao-django-cursor.git meu-app
cd meu-app
rm -rf .git && git init
python -m venv .venv && source .venv/bin/activate
pip install django djangorestframework pytest-django
django-admin startproject config .
```

Ver [`docs/stacks/DJANGO.md`](docs/stacks/DJANGO.md).
