# repo-padrao-jhipster-angular

Template pronto para **JHipster + angular**.

## Clone

```bash
git clone https://github.com/filipefalcaofs/repo-padrao-jhipster-angular.git meu-app
cd meu-app
rm -rf .git && git init
npm install -g generator-jhipster
jhipster
```

Ver [`docs/stacks/JHIPSTER.md`](docs/stacks/JHIPSTER.md).

## Já incluído

- Superpowers + rules universais
- Rules: `jhipster-core`, `jhipster-jdl`, `jhipster-frontend`
- Skills: `jhipster-development`, `jhipster-jdl`, `spring-testing`, `jhipster-angular-development`
- `.jhipster-stack.json` — frontend **angular**
