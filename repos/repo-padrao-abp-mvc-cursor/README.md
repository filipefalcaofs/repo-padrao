# repo-padrao-abp-mvc-cursor

**IDE:** Cursor · **Linguagem:** .NET (C#) · **UI:** mvc

Template **ABP + mvc** — arquivos só para Cursor.

## Clone

```bash
git clone https://github.com/filipefalcaofs/repo-padrao-abp-mvc-cursor.git meu-app
cd meu-app
rm -rf .git && git init
abp new MeuApp -u mvc -d ef
```

Ver [`docs/stacks/ABP.md`](docs/stacks/ABP.md).

Inclui `.mcp.json` — habilitar `abp-studio` no Cursor e abrir a solution no ABP Studio.
