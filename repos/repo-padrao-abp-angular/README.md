# repo-padrao-abp-angular

Template pronto para **ABP + angular**.

## Clone

```bash
git clone https://github.com/filipefalcaofs/repo-padrao-abp-angular.git meu-app
cd meu-app
rm -rf .git && git init
abp new MeuApp -u ng -d ef
```

Copiar `.mcp.json` já incluído; abrir solution no ABP Studio. Ver [`docs/stacks/ABP.md`](docs/stacks/ABP.md).

## Já incluído

- 16 rules oficiais ABP + `abp-mcp`, `abp-ui`
- UI rule: `ui-angular`
- `.abp-stack.json` — UI **angular**
