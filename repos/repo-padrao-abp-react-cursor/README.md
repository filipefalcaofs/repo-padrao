# repo-padrao-abp-react-cursor

**IDE:** Cursor · **Linguagem:** .NET (C#) · **UI:** react

Template **ABP + react** — arquivos só para Cursor.

## Clone

```bash
git clone https://github.com/filipefalcaofs/repo-padrao-abp-react-cursor.git meu-app
cd meu-app
rm -rf .git && git init
abp new MeuApp --template app --modern --ui-framework react
```

Ver [`docs/stacks/ABP.md`](docs/stacks/ABP.md).

Inclui `.mcp.json` — habilitar `abp-studio` no Cursor e abrir a solution no ABP Studio.

## Entrada da IDE

`.cursor/rules/`, `.cursor/skills/`, `.cursor/plugins/superpowers/`
