**IDE:** Codex · **Linguagem:** Python

Template **Django + DRF** — arquivos só para Codex.

## Clone

```bash
git clone https://github.com/filipefalcaofs/repo-padrao-django-codex.git meu-app
cd meu-app
rm -rf .git && git init
python -m venv .venv && source .venv/bin/activate
pip install django djangorestframework pytest-django
django-admin startproject config .
```
