# repo-padrao-jhipster-vue

**Linguagem:** Java (Spring Boot) · **Frontend:** vue

Template pronto para **JHipster + vue**.

## Clone

```bash
git clone https://github.com/filipefalcaofs/repo-padrao-jhipster-vue.git meu-app
cd meu-app
rm -rf .git && git init
npm install -g generator-jhipster
jhipster
```

Ver [`docs/stacks/JHIPSTER.md`](docs/stacks/JHIPSTER.md).
